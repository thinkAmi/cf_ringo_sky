# Ringo Sky

Bluesky の投稿からりんごを集計・可視化するアプリ。りんご品種の系譜（親子関係）と、食べた記録の集計という2つの関心事を扱う。

## Language

**品種**:
りんごの種類。ローマ字 snake_case の正準名 `name`（例: `golden_delicious`）で一意に識別される。
_Avoid_: 銘柄、種類

**表示名**:
品種の日本語表記（例: `つがる`）。UI 表示専用で、識別には使わない。
_Avoid_: 日本語名、ラベル

**系譜**:
ある品種と、その花粉親・種子親の組。品種1つにつき系譜は高々1つ。
_Avoid_: 家系図、親子関係（データの組を指すときは「系譜」で統一）

**花粉親**:
交配で花粉を提供した側の親品種。
_Avoid_: 父、親1

**種子親**:
交配で種子（果実）を実らせた側の親品種。
_Avoid_: 母、親2

**基本品種**:
交配元をたどれない（または調べていない）ため、花粉親・種子親が unknown である品種。
_Avoid_: 原種、ルーツ品種

**枝変わり**:
交配ではなく、既存品種の枝や芽に生じた突然変異（芽条変異）から生まれた品種。系譜は交配の組で表すため、枝変わりは元品種の花粉親・種子親をそのまま引き継ぐ（元品種と同じ親を持つ品種として扱う）。「元品種は◯◯」という由来自体は系譜には載らず、出典に添える（詳細は ADR 0007）。
_Avoid_: 芽条変異、変異種（データ上の扱いを指すときは「枝変わり」で統一）

**unknown**:
親が不明であることを表す番兵品種。基本品種の花粉親・種子親は unknown になる。系譜一覧の表示からは除外される。

**品種マスタ**:
全品種を1品種1行で列挙した一覧。表示名を主キーとし、読み・色・系譜を保持する。系譜を持つのは一部の品種のみ。
_Avoid_: 系譜ファイル、色定義

**出典**:
品種マスタの行の内容（系譜・色・読み）の根拠。Web ページでも書籍でもよい自由記述で、画面には表示しない。
_Avoid_: 参考文献、ソース

**読み**:
品種の呼び方をひらがなで表した、五十音順ソートのためのキー。正式なふりがなではない。英字品種もひらがなで書く。
_Avoid_: ふりがな、ルビ

**フィード**:
Bluesky の `[リンゴ]` タグ付き投稿から抽出した「食べた記録」。品種の系譜とは独立した集計用データ。
_Avoid_: 投稿、ポスト（抽出後のレコードを指す場合）
